# Project Description / 项目描述

**English:** Martingale Multi-Symbol Trading Bot is a Python framework for running martingale-based spot strategies across Binance backtests and testnet sessions. It separates strategy logic, portfolio controls, exchange connectivity, and notifications while supporting dynamic time-decayed take-profit management.

**中文：** Martingale Multi-Symbol Trading Bot 是一个用于 Binance 现货回测与测试网的 Python 马丁格尔策略框架，模块化拆分策略逻辑、资金管理、交易所接口与通知系统，并内建基于时间衰减的动态止盈机制。
